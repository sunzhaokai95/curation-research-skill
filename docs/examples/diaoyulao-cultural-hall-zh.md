# 公开示例:生活方式主题馆资料调研

本示例用于说明新版 skill 的输出形态和审计指标。示例主题为公开、非客户项目的生活方式/兴趣社群类主题馆,不包含真实客户资料、私有路径或未授权项目归档。

## 示例任务

- 主题类型:C 文旅 / 主题馆 / 主题空间
- 地域范围:中国
- 资料目标:围绕一个兴趣社群主题,形成知识型 XMind 和可选 Word 资料报告

## 新版验证重点

- 先研究主题本体,再处理场馆外壳。
- 百科、入门资料、行业资料、学术资料、政策资料共同进入来源层。
- 政府、监管、官方数据库类资料只人工/浏览器读取后记录摘录,不使用脚本抓取。
- 重要概念必须按“是什么 / 不是什么 / 怎么分 / 怎么工作 / 怎么用 / 数据 / 例子 / 来源”展开。
- 输出不写展示形式、展项建议、空间表达、策展主题或观众体验建议。

## 本轮审计指标

- 证据卡:86 条,校验通过。
- 来源:30 个进入证据卡,其中权威/机构/学术/政府类来源 13 个。
- 覆盖审计:全部必查核心通过。
- XMind:2113 个节点,最大层级 7,备注节点 0。
- XMind 污染检查:占位句 0,方法标签 0,字段腔 0,策展污染 0。
- Word 报告:约 2 万字,83 个标题,82 个正文段落,报告审计通过。

## 资料展开方向

示例中的知识型导图覆盖以下方向:

- 主题定义与边界
- 历史源流与阶段变化
- 分类谱系与子主题系统
- 关键对象、工具、材料、文本与影像
- 行为流程、入门路径与进阶路径
- 人物群体、社群组织与代表人物
- 场景空间与日常实践
- 产业消费、平台数据与品牌渠道
- 社会文化、圈层语言与媒介传播
- 政策安全、生态伦理与行业规范
- 最新动态与当前状态
- 来源索引与待核实清单

## 版本意义

这个示例验证了本轮迭代的核心变化:输出不再以证据卡字段为阅读主线,而是以知识传播为主线;证据卡用于事实校准,模型解释层用于把概念讲清楚。
