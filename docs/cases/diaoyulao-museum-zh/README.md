# 案例:钓鱼佬博物馆 / 文化馆资料调研

这是一个公开示例案例,用于展示 `curation-research` skill 的实际产物形态。案例主题属于 C 类:文旅 / 主题馆 / 主题空间,地域范围为中国。

本目录只包含可公开的交付物与审计摘要,不包含本地工作归档、私有路径、未授权项目资料或搜索过程中的中间文件。

| `diaoyulao-museum-research-report.docx` | Word 资料调研报告 |
| `xmind_audit.json` | XMind 审计摘要 |
| `report_audit.json` | Word 报告审计摘要 |

## 调研重点

这个案例验证了新版方法论的几个关键点:

- 先研究“钓鱼 / 钓鱼佬”主题本体,再处理“博物馆 / 文化馆”外壳。
- 资料不只停在网络梗,而是展开到历史源流、钓法分类、钓具系统、行为流程、社群语言、产业消费、政策安全和最新动态。
- 重要概念按“是什么 / 不是什么 / 怎么分 / 怎么工作 / 怎么用 / 数据 / 例子 / 来源”展开。
- 政府与政策类公开来源使用人工/浏览器读取后记录摘录,不通过脚本抓取。
- 输出不包含展项建议、展示形式、空间表达、策展主题或观众体验建议。
